# Customer Details Form

## Project Description
This project is a simple **Customer Details Form** created using **HTML, CSS, and JavaScript**.  
The form collects customer information and checks whether the entered data is correct using JavaScript validation.

After successful validation, the customer details are displayed on the same webpage.

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## Features
- User can enter personal details.
- Input validation is applied to all required fields.
- Error messages are shown if wrong data is entered.
- Form does not submit until all validations are correct.
- Entered data is displayed after successful submission.

---

## Validations Included
- First Name and Last Name → only letters allowed
- Address → cannot be empty
- City and State → letters only
- Pincode → must be 6 digits
- Phone Number → must be 10 digits
- Email → valid email format required
- Dropdown selection required
- Checkbox must be selected
- Gender selection required
- Table fields must be filled correctly

---

## How to Run the Project
1. Download the project folder.
2. Open the folder.
3. Open `index.html` in any web browser.

No installation is required.
